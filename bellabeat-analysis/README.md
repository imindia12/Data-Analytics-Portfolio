# Bellabeat Case Study
## Project Overview
This project is a capstone case study completed as part of the Google Data Analytics Professional Certificate. The goal was to analyze smart device usage data to identify trends in user behavior and provide data-driven marketing recommendations for Bellabeat, a high-tech company that develops health-focused products for women.

## Business Task
Analyze Fitbit smart device data to uncover patterns in activity, sleep, and wellness behaviors, and use those insights to inform Bellabeat’s marketing strategy.

## Data Sources
Fitbit Fitness Tracker Data (public dataset)
Multiple CSV files containing daily activity, sleep, and weight information
Note: Data was anonymized and provided for educational purposes.

## Tools Used
* Google Sheets – data cleaning, aggregation, and analysis
* Pivot tables & formulas – trend analysis and summaries
* Charts & tables – data visualization
* Data Cleaning & Preparation
* Removed duplicates and irrelevant records
* Standardized column names and formats
* Joined datasets using a common user ID
* Verified data consistency and completeness
  
## Key Insights
* The analysis of user activity, sleep, and weight data revealed clear patterns in daily habits, physical activity, and health metrics​​
* Users are most active Wednesdays and Saturdays​​
* Sleep patterns vary throughout the week, showing potential opportunities for personalized wellness recommendations​​
* Higher weight correlates with higher BMI
​
## Recommendations
Increase personalized reminders on lower activity days​
  - data showed that users are least active Thursdays. Consider challenges,   reminders, or targeted app notifications mid-week.​

Market holistic wellness tracking​
  - Increase awareness of correlation between activity, sleep, and weight to overall wellness.​

Promote high-value features on peak days​
  ​- Users are most active on Wednesdays and Saturdays. Highlight tracking features, wellness tips, or premium content on these days.
